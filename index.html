Colors

To incorporate a new color, add the color variable to the variables it in the Sass map inside the colors partial.

Spacing

                

                    
Use variables or classes for spacing.

                    
Spacing helpers cam be used for padding, margins, positioning elements, etc.

                    
The spacing variables are   $spacing, $spacing-2, $spacing-3, and $spacing-half.

                    
The default  $spacing variable's value is 10 pixels.

                    
 $spacing-2 is twice the value of $spacing and so on.

                

                

                    
Reference files: _globals.scss, _spacing.scss, _variables.scss
